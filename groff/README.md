PAPERSIZE
=========

As of groff version 1.22.2 release 2 the default papersize for groff
can be defined in `/etc/papersize`.  Just add the required papersize
like 'letter' or 'a4' here, default is 'letter'.

See the following URL for more information:
http://www.gnu.org/software/groff/manual/html_node/DESC-File-Format.html
